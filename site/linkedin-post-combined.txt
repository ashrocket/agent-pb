I keep building tiny tools because I'm too lazy to do things manually.

The pattern is always the same: the AI agent does something smart, then I have to drag a file, select some text, copy, switch windows, paste. The agent has shell access — why am I the one doing this?

So I built two tools:

agent-look — say /look and the agent scans your recent screenshots, renames them from "Screenshot 2026-03-23 at 2.15.32 PM.png" to something useful, and pulls them into context. No dragging. It even uses macOS Spotlight OCR to avoid burning vision tokens.

agent-pb — say /pb and the agent finds the most recent useful artifact in your conversation (query, command, code block, URL) and puts it in your clipboard. No selecting, no highlighting. Just paste.

Both are Claude Code plugins, but the prompts are portable. Drop them into Cursor, Windsurf, Copilot — anything that can run a shell command. macOS only for now — Spotlight OCR and pbcopy are hard dependencies.

Same philosophy: laziness as a design principle. The agent is right there with shell access — let it handle the boring parts.
